>  ESlint config  for Airbnb-Base 、Jest 、TypeScript、Prettier


## Install

```node

npm i eslint-config-ajtp -save-dev
// or
yarn add eslint-config-ajtp -D

```

## Usage

.eslintrc
```js
{
  "extends": ["ajtp"]
}

```
